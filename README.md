# 75 Hard

A tiny in-browser tracker for my version of the challenge in an attempt to follow it :( No accounts and your progress lives in this browser’s `localStorage` (so don’t expect it to follow you to your phone lol)

## Run it

```bash
cd 75H
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080). Or double-click `index.html` like it’s 2008 — that works too, just use the same `file` vs `http` origin or your saves won’t look like the same “site”.

## Stack
HTML, CSS, JavaScript.
